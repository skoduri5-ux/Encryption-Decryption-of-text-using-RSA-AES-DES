# Encryption-Decryption-of-text-using-RSA-AES-DES

# RSA (Rivest–Shamir–Adleman) – it is Asymmetric Encryption.
Key Sizes:
Common: 1024 bits, 2048 bits, 3072 bits, 4096 bits.
# Recommended Minimum: 2048 bits
Security Note: Larger keys = stronger encryption, but slower performance. RSA uses two keys: public and private.

# AES (Advanced Encryption Standard) – it is Symmetric Encryption
Key Sizes:
128 bits (AES-128)
192 bits (AES-192)
256 bits (AES-256)
Block Size: Always 128 bits
Security Note: AES-256 offers the highest security and is widely used in modern applications, including government and military.

# DES (Data Encryption Standard) – it is Symmetric Encryption
Key Size: 56 bits
Block Size: 64 bits
Security Note: Considered insecure today due to short key size—can be brute-forced in a short time.
